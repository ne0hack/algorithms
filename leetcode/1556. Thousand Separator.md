# 1556. Thousand Separator


## Description
Given an integerÂ `n`, add a dot (".") as the thousands separator and return it in string format.

#### Example 1:
```
Input: n = 987
Output: "987"
```

#### Example 2:
```
Input: n = 1234
Output: "1.234"
```


## Solution

### Python3
```python
class Solution:
    def thousandSeparator(self, n: int) -> str:
        result = []
        n = str(n)[::-1]
        for i in range(0, len(n), 3):
            result.append(n[i:i+3][::-1])
        return ".".join(result[::-1])
```
