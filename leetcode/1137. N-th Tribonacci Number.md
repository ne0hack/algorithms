# 1137. N-th Tribonacci Number

## Description
The Tribonacci sequence Tn is defined as follows:Â 

T0 = 0, T1 = 1, T2 = 1, and Tn+3 = Tn + Tn+1 + Tn+2 for n >= 0.

Given `n`, return the value of Tn.

#### Example 1:
```
Input: n = 4
Output: 4
Explanation:
T_3 = 0 + 1 + 1 = 2
T_4 = 1 + 1 + 2 = 4
```

#### Example 2:
```
Input: n = 25
Output: 1389537
```


## Solution

### Python3
```python
class Solution:
    def tribonacci(self, n: int) -> int:
        if n == 0: return 0
        t_seq = [0, 1]
        while len(t_seq) != n+1:
            t_seq.append(sum(t_seq[-3:]))
        return t_seq[-1]
```
