# Basics of Sets and Relations #1


## Description
You are given two sets.\
Set **A** = {1,2,3,4,5,6}\
Set **B** = {2,3,4,5,6,7,8}\
How many elements are present in _A U B_?\
Only enter the correct integer in the editor below. Do not include any extra spaces, tabs or newlines.


## Solution
```
8
```
